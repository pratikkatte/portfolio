# Pratik Katte Portfolio

Personal portfolio for Pratik Katte, built with Next.js and prepared for Vercel.

## Local setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
```

## Vercel

Import this folder or its GitHub repository into Vercel as a Next.js project.
The default build command is `npm run build`.

After the project is deployed, add `pratikkatte.com` in Vercel project settings
under Domains and follow the DNS records Vercel shows for your registrar.
